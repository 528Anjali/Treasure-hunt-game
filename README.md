#  Treasure Hunt Game

##  Description
This is a web-based Treasure Hunt Game developed using the MERN stack. The game consists of multiple levels where users solve riddles to progress.

##  Features
- Multi-level gameplay
- Score tracking
- Hint system
- Sound effects
- Animations
- Leaderboard

##  Technologies Used
- React.js
- Node.js
- Express.js
- MongoDB

##  How to Run
1. Clone the repository
2. Run backend:
   cd server
   npm start
3. Run frontend:
   cd client
   npm start
